# Specs

This directory contains feature specifications for the EmprendYup platform.

## Naming Convention

```
specs/<feature-name>.md
```

## Spec Template

```markdown
# Spec: <Feature Name>

## Goal

One sentence describing what this feature does.

## User Stories

- As a <customer|seller|admin>, I want to <action> so that <benefit>.

## Requirements

- Requirement 1
- Requirement 2

## Acceptance Criteria

- [ ] Criterion 1
- [ ] Criterion 2

## Business Type Scope

- Applies to: store | restaurant | service | event | all

## Out of Scope

- What this feature does NOT include.
```

## Archive

Completed specs are moved to `specs/archive/`.
