## How to Use This File

This file always points to the **highest-priority incomplete task** for the development team.

To pick up work:

1. Read the task file listed above (in `tasks/`)
2. Read the referenced spec in `specs/`
3. Read `docs/ARCHITECTURE.md` for technical guidelines
4. Implement the task following the relevant agent's prompt in `agents/`
5. When done, update this file to point to the next task

---

## Getting Started

To create your first task:

1. The **Product Agent** creates a spec in `specs/`
2. The **Product Agent** creates a task file in `tasks/`
3. Update this file to point to that task

**Example:**

```
Current Task: tasks/01-product-search-improvements.md
Spec: specs/product-search.md
Agent: frontend-agent
```
